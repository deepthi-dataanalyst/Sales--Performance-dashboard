# 📊 Sales Performance Dashboard

## 📌 Project Overview
A 12 month Sales vs Target Performance Dashboard built using Microsoft Excel.

## 🛠 Tools Used
- Microsoft Excel

## 📐 Formulas Used
- SUM - To calculate Total Sales and Total Target
- IF - To check Achieved or Not Achieved
- Achievement % = Total Sales / Total Target x 100

## 📈 Results
- Total Sales: 5,68,000
- Total Target: 5,00,000
- Achievement Rate: 113.6%

## 📊 Dashboard Features
- 12 month Sales vs Target Bar Chart
- Colour coded Status column
- Achievement percentage calculation
